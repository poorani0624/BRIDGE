# 🌉 BRIDGE

BRIDGE is a Flask-based e-commerce platform that connects customers with authentic Indian products through a seamless online marketplace. The platform enables buyers, sellers, logistics personnel, and administrators to manage the complete product ordering and delivery process.

## ✨ Features

- 👤 User, Seller, Logistics & Admin authentication
- 🛍️ Product browsing and category-wise listing
- 🛒 Shopping cart and checkout
- 📦 Order tracking with logistics updates
- 🏪 Seller dashboard for product management
- 🛠️ Admin dashboard for platform management

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, Jinja2
- **Backend:** Python, Flask
- **Database:** SQLite

## 🚀 Getting Started

```bash
git clone https://github.com/poorani0624/BRIDGE.git
cd BRIDGE
pip install -r requirements.txt
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
